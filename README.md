# Draw-100-Circles

# Description
This project is about a fish bowl like circle where different size circles will appear in water. You can reset it, change the speed, and make it appear in the dark. Their is also a tracker to count the number of circles and stops when hit 100.

# Features
- Reset button
- Glow in the dark button
- Speed toggle
- Circle counter
